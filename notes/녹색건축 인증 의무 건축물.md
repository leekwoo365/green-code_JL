# 녹색건축 인증 의무 건축물

## 근거
- 녹색건축물 조성 지원법 제16조⑦
- 같은 법 시행령 제11조의3

## 의무대상 요건 (4가지 모두 충족)

| 요건 | 기준 |
|------|------|
| 소유/관리 주체 | 중앙행정기관·지방자치단체·공공기관·교육기관의 장 또는 교육감 |
| 건축 범위 | 신축·재축 또는 별동 증축 (기존 대지에 별개 건축물로 증축) |
| 연면적 | **3,000㎡ 이상** (복수 건축물 시 합산) |
| 에너지 절약계획서 | 법 제14조①에 따른 제출 대상일 것 |

- **용도 제한 없음** — 업무시설, 병원, 학교, 문화시설 등 용도 무관
- 민간 건축물은 의무 아님 (자발적 인증 가능)
- 리모델링·대수선은 해당 없음

## 의무 인증 등급

| 구분 | 의무 등급 |
|------|----------|
| **공공업무시설** (건축법 시행령 별표1 제14호가목) | **우수 (그린 2등급)** 이상 |
| **그 외 공공건축물** | **일반 (그린 4등급)** 이상 |

> 4등급이 최하 등급이므로, 공공업무시설 외에는 사실상 인증만 받으면 됨

## 녹색건축 인증 등급 체계

| 등급 | 명칭 |
|------|------|
| 그린 1등급 | 최우수 |
| 그린 2등급 | 우수 |
| 그린 3등급 | 우량 |
| 그린 4등급 | 일반 |

## 미이행 시
- **2천만원 이하 과태료** (법 제41조①제7호)

## 혼동 주의 — 용도 제한이 있는 것은 다른 조항
- **차양·건축설비 설치 의무** (법 제14조의2, 영 제10조의2)는 공공 + 3,000㎡ 이상 + **업무시설 또는 교육연구시설만** 해당
- 녹색건축 인증 의무 (영 제11조의3)는 **용도 무관**
